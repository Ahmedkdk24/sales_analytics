# Sales Analytics

This repository contains sales analytics tools, including churn analysis and supporting scripts.

## Files

- `churn_analysis.py`: Perform churn analysis and generate output reports.
- `requirements.txt`: Python dependencies for the project.

## Usage

Install dependencies and run the analysis script:

```bash
pip install -r requirements.txt
python churn_analysis.py --input combined.xlsx --out churn_analysis.png
```
